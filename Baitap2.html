What is CSS

Cascadings Style Sheets (CSS) is used to format the layout of a webpage

With CSS, you can control the color, font, the size of the text, the spacing between elements, how elements are positioned and laid out, what background images or background colors are to be used, different displays for different devices and screen sizes, and muc more!

Tips: The word cascadings means that a style applied to a parent element will also apply to all children elements within the parent, so if you set the color of the body text to "blue", all headings, paragraphs, and other text elements within the body will also get the same color (unless you specify somthing else)
